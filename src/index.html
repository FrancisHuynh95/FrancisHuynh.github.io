Projects

AudioDome is inspired by Spotify. It allows creators to upload their music, and interact with the community. This was a group project created with JavaScript, React.js, Redux.js for the frontend, Python, Flask and PostgreSQL for the backend, as well as AWS to handle image and audio uploads. Users are able to upload, delete, edit, and listen to music, as well as create, update, delete, and view playlists. Once an account has been created, the user is able to edit the account, as well as delete it.
